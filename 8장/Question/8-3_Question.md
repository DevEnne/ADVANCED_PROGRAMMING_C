- (AP2023) 8장 (3) 소문자와 대문자의 등장 횟수 (p. 308)
## 문제 설명
문자를 입력 받은 후 알파벳 문자의 대문자와 소문자의 등장 횟수를 `‘*’`로 나열하는 그래프를 출력하라.
교재 p. 308 문제와 다르게 `'\n'`이 입력되었을 때 `while loop`에서 나오게 작성하자.

입력 코딩 예시:
```c
char ch;
while( (ch = getchar()) != '\n') {
  ...
}
```

## 입출력 예시 1
`Alphabet Only!`  
```
대문자**
소문자**********
```

## 입출력 예시 2
`Java VS C++`  
```
대문자****
소문자***
```

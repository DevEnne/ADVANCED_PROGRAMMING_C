```c
int main()
{

 if (  == NULL)
        printf("NO.");
    else
        printf("이전문자는 %s입니다.", p);
}
```

## 입력 예시 1
`STRING I`

## 출력 예시 1
`이전문자는 STRI입니다.`

`Halloween o`

## 출력 예시 2
`이전문자는 Hallo입니다.`

`Pororo O`

`NO.`
